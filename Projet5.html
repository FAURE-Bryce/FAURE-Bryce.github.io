Projet de stage n°5

Ce projet consistait à réaliser une application clientlourd en C# en groupe. Un congre de sport et de santé nous a fourni un cahier des charges consistant à créer une application de gestion de ce dit congre. Nous avons donc réalisé une base de données SqlServer, le script de création se trouve dans le fichier et se nomme « gestionOrganisationCongres.sql ». Pour ce projet, j’ai réalisé une partie de la base de données, plus précieusement les procédures stockées « NbPlacesBySession », «nbPlacesActivite » et le trigger pour l’insertion dans la table "Participer". Pour la partie de l’application, j’ai réalisé une grande partie de la partie graphique de tout le formulaire et j’ai également réalisé entièrement le formulaire des sessions :

La page accueil

La page d'accueil présente les différentes pages, dont la gestion de Session.

Page des sessions

Tout d'abord, cela affiche la liste de toutes les informations de toutes les sessions dans un dataGrid. Il y a également un bouton "Ajouter une session" qui ajoute une nouvelle session.

Page de détail d'une session

La page de détail d'une session permet d'afficher et de modifier les informations d'un la session sélectionné dans le dataGrid de la page précédente.

Page d'ajout de session

La page d'ajout de session permet d'ajouter une nouvelle session.
